# 北京链家二手房数据集说明

## 变量说明

### 小区数据 community_describe.csv

1. index	索引
2. id		小区ID
3. community	小区名称
4. district		区
5. bizcircle		街道
6. tagList		标签列表
7. onsale		在售房源数量

### 房源数据 house.csv

1. index	索引
2. title		房屋标题
3. community	小区名（关联字段）
4. years	房屋年限信息
5. housetype	房屋类型
6. square	平米数
7. floor	层数
8. taxtype	契税信息
9. totalPrice	总价格（万元）
10. unitPrice	单价（元）
11. followInfo	带看信息
