# 泰坦尼克号数据集说明

数据集收集了在泰坦尼克号撞山事件中涉及到的人员信息，以此信息希望分析得到某些结论，从而通过某人的特征信息判断人员生存的概率。

## 变量说明

### 训练集数据 train.csv

1. PassengerId => 乘客ID
2. Survived => 生存
3. Pclass => 乘客等级(1/2/3等舱位)
4. Name => 乘客姓名
5. Sex => 性别
6. Age => 年龄
7. SibSp => 堂兄弟/妹个数
8. Parch => 父母与小孩个数
9. Ticket => 船票信息
10. Fare => 票价
11. Cabin => 客舱编号
12. Embarked => 登船港口
